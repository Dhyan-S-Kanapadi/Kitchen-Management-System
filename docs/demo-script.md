# Demo Script

1. Start backend and frontend.
2. Open Dashboard and explain the OS-to-kitchen mapping.
3. Click Demo and load the scenario.
4. Use the Demo page to show process fields, queues, Gantt chart, metrics, context switches, and decision explanation.
5. Open Resources and show semaphore value changes.
6. Open Synchronization and show producer-consumer plus mutex critical section.
7. Open Deadlock and show the resource allocation cycle.
8. Open Analytics and Report to summarize metrics.
