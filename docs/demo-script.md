# Demo Script

1. Start backend and frontend.
2. Open Dashboard and explain the OS-to-kitchen mapping.
3. Click Demo and load the scenario.
4. Open Orders and show process fields: arrival time, cooking time, priority, resources, ingredients, and recipe pages.
5. Open Scheduler and run FCFS, SJF, Priority, and Round Robin.
6. Point to Gantt chart, metrics, context switches, and decision explanation.
7. Open Resources and show semaphore value changes.
8. Open Synchronization and show producer-consumer plus mutex critical section.
9. Open Deadlock and show the resource allocation cycle.
10. Open Analytics and Report to summarize metrics.
